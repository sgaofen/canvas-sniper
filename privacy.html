Canvas Sniper - Privacy Policy

Last updated: December 2024

Overview

Canvas Sniper ("the Extension") is committed to protecting your privacy. This policy explains how we handle your data.

Data Collection

The Extension collects and processes the following data:

  • Page Screenshots: Temporarily captured to analyze quiz questions. These are sent to Google's Gemini API for processing and are not stored.
  • Question Text: Extracted from the page to assist with answer analysis.
  • User Preferences: Settings (enabled state, stealth mode, cursor status) are stored locally in your browser.

Data Usage

  • Screenshots and text are used solely for analyzing quiz questions
  • Data is sent to Google's Gemini API for AI processing
  • No data is stored on our servers
  • No personal information is collected

Third-Party Services

This extension uses Google's Gemini API. Your use of this extension is also subject to Google's Privacy Policy.

Data Retention

The Extension does not retain any user data. All processing is done in real-time and data is discarded immediately after use.

User Rights

You can:

  • Disable the extension at any time
  • Clear local storage to remove preferences
  • Uninstall the extension to remove all associated data

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date.

Contact

For questions about this privacy policy, please contact us through the Chrome Web Store listing.
