## 简短描述 (132字符以内)

## 详细描述

🎯 Canvas Sniper - Canvas 智能答题助手

让 Canvas LMS 测验变得更轻松！只需双击题目，即可自动识别并填写答案。

✨ 主要功能：
• 智能识别 - 自动识别下拉选择题、填空题、选择题等多种题型
• 一键答题 - 在题目区域双击鼠标即可触发自动答题
• 无痕模式 - 隐藏状态提示，操作更加隐蔽
• 鼠标状态 - 可选的加载光标显示
• 内置帮助 - 完整的使用说明

🚀 使用方法：
1. 打开 Canvas 测验页面
2. 确保扩展已启用
3. 在题目区域双击鼠标
4. 等待自动填写答案

⌨️ 快捷键：
• ESC - 取消正在进行的分析

⚙️ 支持网站：
• Canvas LMS (*.instructure.com)
• 教育机构网站 (*.edu)

⚠️ 注意事项：
• 确保题目完全显示在屏幕内
• 答案仅供参考，请自行核实正确性
• 请遵守所在学校的学术诚信政策

## 上传步骤

1. 访问 https://chrome.google.com/webstore/devconsole
2. 注册开发者账号（$5 一次性费用）
3. 点击 "New Item"
4. 上传 ZIP 文件
5. 填写上述描述信息
6. 上传截图
7. 填写隐私政策 URL（上传 privacy.html 到你的网站或用 GitHub Pages）
8. 提交审核

审核通常需要几天时间。
